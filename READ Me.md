# Passive vs. Active Attack Classifier Lab

## 📌 Overview
This hands-on cybersecurity lab demonstrates how to use Machine Learning (ML) to classify network threats. Participants will build and evaluate a supervised ML model to differentiate between **Passive Attacks** (e.g., sniffing, traffic analysis) and **Active Attacks** (e.g., DoS, data modification).

---

## 🎯 Learning Objectives
By the end of this lab, you will be able to:
* Understand the conceptual differences between passive and active network threats.
* Preprocess cybersecurity network traffic datasets using Python.
* Train a classification model (like Random Forest or Decision Tree) using `scikit-learn`.
* Evaluate the model's accuracy and performance.

---

## 🛠️ Tech Stack & Prerequisites
* **Language:** Python 3.x
* **Libraries:** `pandas`, `numpy`, `scikit-learn`, `matplotlib`, `seaborn`
* **Environment:** Jupyter Notebook / Google Colab / GitHub Codespaces
