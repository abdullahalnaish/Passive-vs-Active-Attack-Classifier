# Passive vs Active Attack Classifier

A practical lab to build a machine learning model that classifies network attacks into Passive (like sniffing) or Active (like DoS).

## What you will learn
* Preprocessing network traffic data.
* Training a Scikit-Learn classifier.
* Evaluating model performance (Accuracy & Confusion Matrix).

## Requirements
* Python 3.x
* pandas, numpy, scikit-learn, matplotlib
